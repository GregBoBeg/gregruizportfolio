Application Overview

NoNoMedia.net is a news aggregator and political-movement website that empowers readers to hold journalists accountable for the articles they publish. NoNoMedia gives readers the ability to vote and comment on articles published by journalists from other news organizations. Each month, votes are tallied to determine the winning authors of the most trusted and least trusted news articles.

Tech Stack

Behind The Scenes

NoNoMedia's architecture forges modern design concepts into a sophisticated solution for readers and content providers alike. This full-stack project has a Python/Django backend running on a PostgreSQL database, with the data served up through Django REST APIs to a React front-end.

The application leverages a customized Django admin implementation for managing content such as news articles, determining winning authors, and managing the overall application presentation. A custom User model provides account logins using email and unique usernames for online identity purposes. Extensive application security is accomplished by with Django, Django REST and Django Knox security capabilities, along with a middleware solution I developed to enable HTTPOnly cookie storage in the browser.

The frontend, is a single page application (SPA) constructed in JS/React, with a clean, responsive user interface. It includes a full login process that enforces authentication-required functionality, along with an email verification system for account activation. Registered users are empowered to vote and comment on articles. Votes and comments are tracked on a per-article, per-user basis for and are served up via complex database queries to form an elegant mix of both detailed tracking and ease-of-use. Complex logic and queries are handeled with a customized views and serializers to deliver simplicity throughout the user experience. The site also includes an on-going photo library that allows recycling prior photos for future news articles.
